# Affirmations App

## 📱 Overview
Affirmations is a simple Android application that displays a list of motivational affirmations using **Jetpack Compose and Recycler-like layouts**.

## ✨ Features
- Scrollable list of affirmations
- Image and text combination
- Efficient list rendering

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## 📂 Project Structure
- `MainActivity.kt`
- `DataSource.kt`
- `Affirmation.kt`

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Run on an emulator or device

## 📚 Learning Outcome
This project demonstrates how to implement **LazyColumn lists and data-driven UI components** in Jetpack Compose.
